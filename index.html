Google Store

Email Developer I/II / QA Analyst

Upon commencing my employment with Hawkeye, I was fortunate enough to exclusively handle Google emails. However, as my responsibilities grew over time, I gradually took on the task of overseeing Hawkeye's campaign partials for the Google Store wing.

During my tenure, my duties entailed analyzing campaign-specific Figma/Sketch files to create reusable Handlebars and Jinja partials, developing and testing pixel-perfect emails on mobile devices and email testing platforms like Litmus, and performing code reviews for my Hawkeye colleagues.

In the latter stages of my employment, I collaborated solely with Google Store engineers to generate US + rest-of-world AMP emails that incorporated various engaging features for users. Our combined efforts resulted in the successful deployment of interactive emails, which reached a vast audience of millions across the globe.

As a QA analyst, I was responsible for drafting detailed checklists for new HTML emails, landing pages, and plaintext, which involved analyzing HTML/CSS using developer tools to ensure pixel-perfection across multiple email client platforms, and logging bugs through Workfront with concise wording and descriptive screenshots to detail any issues found during testing.

HTML5, CSS/LESS/SCSS, Freemarker, Jinja, Handlebars.js, YAML, JSON, Litmus, Figma, Sketch, Google AMP, Workfront, Jira, A11y Accessibility Compliance, Chrome Developer Tools, Quality Assurance, Google API, Git

Nest Renew

Email Developer II

Utilizing the knowledge and skills I gained while working on Google Store emails and partials, my manager entrusted me with the responsibility of assisting the Nest Renew client in creating a reusable set of accessible modules and templates for their forthcoming eco-friendly email campaign updates.

Being the sole developer of this project, I developed reusable email partials utilizing a combination of Handlebars.js and the client's Connect Composer snippet platform. Subsequently, over the following few months, I produced and dispatched several emails to recipients in the United States and Spain.

Over time, I trained additional Hawkeye developers to work on the campaign and produced detailed documentation concerning the frequently used partials and snippets.

HTML5, CSS/LESS/SCSS, Freemarker, Jinja, Handlebars.js, YAML, JSON, Litmus, Figma, Sketch, Google AMP, Workfront, Jira, A11y Accessibility Compliance, Chrome Developer Tools, Google API, Git

Tci

Intern Developer

As a budding intern I primarily worked on TCI's primary web presence -- specifically the curriculum pages.

During my internship I implemented web accessibility compliance WCAG 2.0 to existing Rails views using core front-end technologies, created modular components with JS libraries, such as React and jQuery, ensured code quality and longevity through unit tests using RSpec and FactoryGirl, and set up CSV generation using Rails helpers, Sidekiq workers, and application PostgreSQL DB data.

For larger tasks such as implementing new web pages, I pair programmed with the primary database manager to pull in CSV / website data to properly populate curriculum pages.

Ruby on Rails, React, jQuery, Javascript, PostgreSQL, MongoDB, HTML5, CSS/LESS, JSON, Git
